# Lore Story Architecture

This folder is the canonical planning space for `LORE`.

## Canonical docs

- [full-story-structure.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/architecture/full-story-structure.md): Stable world lore, character foundations, themes, and the master sequence map.
- [act-1-outline.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/architecture/act-1-outline.md): Fast working outline for Act 1.
- [current-handoff.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/architecture/current-handoff.md): The exact current story position and the immediate next beat.
- [story-rules.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/architecture/story-rules.md): Guardrails for tone, reveals, character use, and collaboration.

## Supporting canon docs

- [world-lore.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/world-lore.md): Kingdom structure, religion, lineage, crystal lore, exile policy, and setting-level truths.
- [political-motives.md](C:/Users/leoja/Desktop/Dad Games/lore-project/story-documents/political-motives.md): Agendas, hidden motives, and political stakes for rulers, rebels, and institutions.

## Character canon

- Character-specific canon lives under `characters/<name>/character.md`.
- Use those files for public perception, hidden truth, relationships, locked canon, and TBD questions for each major figure.

## Workflow

- Update `current-handoff.md` whenever the active working position changes.
- Update `act-1-outline.md` when beat order or pacing changes.
- Update `full-story-structure.md` when canon changes.
- Update `story-rules.md` when we lock a storytelling constraint or collaboration rule.
- Update `world-lore.md` when kingdom, religion, prophecy, crystal, or setting truths change.
- Update `political-motives.md` when a character or institution's agenda becomes clearer.
- Update character files when new personal, relational, or hidden-character canon is confirmed.

## Legacy docs

The original `.txt` files are preserved for reference, but the Markdown files in this folder are the preferred source of truth going forward.
