## Purpose

Use these documents for:
- prose-form story memory
- scene-feel and emotional continuity
- narrative handling of canon that is harder to capture in outline form
- maintaining parallel prose tracks when they serve different creative functions

## Authority

- These documents can be canon.
- `.txt` status does not make them non-canon.
- Their role is different from compressed reference docs, not lower in importance.

## Relationship to other folders

- Use `story/world/world-lore/` for compressed world-reference truth.
- Use `story/world/story-guides/` for sequence order, structure, and timeline handling.
- Use `story/characters/` for per-character canon.

## Note on parallel prose tracks

- Multiple prose tracks may exist in parallel when they serve different purposes.
- A more novelized version and a more screenplay-facing version can both remain active if they are both being intentionally maintained.
- If one track becomes obsolete, mark that status explicitly rather than assuming age or file format communicates it.
