# Lore Story Guides

This folder holds guide documents for story shape, sequencing, and chronology.

These are not just worldbuilding notes. They are the documents that organize how the canon plays out across the story.

## Files here

- [full-story-structure.md](/C:/Users/leoja/Desktop/Dad%20Games/lore-project/story/world/story-guides/full-story-structure.md): master story map for backstory, reveals, placements, and major sequence order
- [act-1-outline.md](/C:/Users/leoja/Desktop/Dad%20Games/lore-project/story/world/story-guides/act-1-outline.md): fast sequence outline for the current Act 1 material
- [official-lore-timeline.txt](/C:/Users/leoja/Desktop/Dad%20Games/lore-project/story/world/story-guides/official-lore-timeline.txt): chronological event order for the broader canon

## What belongs here

- act and sequence planning
- reveal placement
- master structure
- canon chronology
- story-level arrangement docs that connect character and world truth into narrative order

## What does not belong here

- per-character canon
- purely world-reference docs
- active session handoff notes
- adaptation-specific shot and production planning

Those belong in:
- `story/characters/`
- `story/world/world-lore/`
- `story/for-agents/`
- `film/`

## Authority

- Use these docs when the question is “how does the story unfold?”
- Use `story/world/world-lore/` when the question is “what is true about the world?”
- Use `story/characters/` when the question is “what is true about this person?”

## Workflow

- Update `full-story-structure.md` when larger canon placement or reveal logic changes.
- Update `act-1-outline.md` when the active early-story sequence order changes.
- Update `official-lore-timeline.txt` when chronology shifts or gets clarified.
- When one of these changes, check `story/for-agents/current-handoff.md` and related rules for drift.
