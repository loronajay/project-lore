# Lore Story Docs

This folder is the main source-of-truth home for story material in `LORE`.

It is organized by role:
- `characters/` for per-character canon and art reference
- `world/` for world canon, story guides, and prose-form canon
- `for-agents/` for fast operational docs used during active writing and canon maintenance

`film/` sits alongside this folder as adaptation-facing material derived from the story docs here. It should not become the primary place where canon is invented unless a film-specific change has already been intentionally chosen.

## Folder map

### `characters/`

Character-by-character canon.

Use this for:
- `character.md`
- `summary.md`
- concept art grouped by character

### `world/`

Shared world, structure, and prose canon.

Use this for:
- worldbuilding and political truth
- story arrangement and timeline docs
- prose-form canon documents such as screenplay prose tracks

### `for-agents/`

Fast-load operational docs for active work sessions.

Use this for:
- handoff state
- rules and guardrails
- quick canon primers
- agent-facing summaries that help maintain continuity

## Authority

- The story source of truth lives in this folder.
- File format does not determine canon status. Both Markdown and `.txt` files may be canon.
- When documents disagree, prefer the most intentionally maintained canon doc for that topic rather than assuming Markdown always wins.

## Working rule

Before moving or renaming files again, make sure:
- the destination folder has a clear README
- dependent references in `film/` and the workbench are updated in one sweep
- the new location makes the file's role more obvious, not less
