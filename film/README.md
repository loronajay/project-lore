# Film Workspace

This folder is the production-facing home for turning `LORE` into a visual film.

The rest of the repository remains the source-of-truth space for:
- worldbuilding
- canon
- character development
- screenplay-facing story evolution

This folder exists to translate that material into a film production pipeline.

## Purpose

Use this folder for:
- adaptation planning
- visual development
- shot and sequence planning
- solo production workflow
- AI tool strategy
- asset tracking

Do not use this folder as the primary place to invent canon unless a film-specific change has already been intentionally chosen.

## Relationship to the rest of the repo

- `story/` remains the main writing and story-reference area.
- `story/characters/` remains the main character canon and art-reference area.
- `world-art/` remains environment and world visual reference.
- `film/` converts those materials into a producible movie plan.
