# Tool Stack

## Core recommendation

Use a stack built around directed asset creation, not one-click full-movie generation.

## Primary categories

### Image generation

Use for:
- character sheets
- location keyframes
- mood frames
- prop design
- costume exploration

Possible tools:
- Midjourney
- ChatGPT image generation
- other high-quality still-image generators as preferred

### Video generation

Use for:
- short image-to-video motion shots
- environmental movement
- dream fragments
- inserts and transitions

Possible tools:
- Runway
- Pika
- other short-form cinematic video generators

### Voice

Use for:
- temp dialogue
- narration tests
- scratch performance passes
- possibly final stylized voice work

Possible tools:
- ElevenLabs
- manual recorded temp voice if preferred

### Editing and assembly

Use for:
- timeline building
- sound design
- temp score placement
- color shaping
- final export

Recommended:
- DaVinci Resolve

Optional:
- Descript for early rough assembly and script-driven editing experiments

## Working principle

Each tool should have one main job.

Do not depend on any single platform to:
- write
- storyboard
- generate all visuals
- animate all shots
- perform all editing

## Immediate practical stack

If keeping cost moderate:
- one image tool
- one video tool
- one voice tool
- DaVinci Resolve
