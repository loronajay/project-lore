# Asset Plan

## Purpose

Track the reusable assets needed to build the film efficiently.

## Asset groups

### Character packages

Each major character should eventually have:
- front-facing reference
- side and three-quarter reference
- expression range
- clothing guide
- age/condition variants if needed
- lighting reference

High priority:
- Leo
- Richard
- Marshe
- Leo's father

Secondary priority:
- Esmeralda
- The King
- villagers

### Environment packages

High-priority early environments:
- Leo's house
- market square
- alley where Marshe corners Leo
- path network through village
- exterior of Richard's house
- Richard's hidden workspace

### Prop packages

High-priority props:
- mother's jewelry box
- dish
- bread
- unfinished sword
- pole/training stick

### Dream and memory packages

Need visual libraries for:
- battlefield fragments
- crystal imagery
- mounted general image language
- warm family market memory
- father training memory

## Asset-building rule

Before generating many shots for a sequence, lock:
- character look
- costume
- environment mood
- key props
