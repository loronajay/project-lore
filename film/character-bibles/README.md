# Character Bibles

This folder holds film-production character bibles.

These are not meant to replace the canon docs in `story/characters/`.

## Purpose

Use character bibles to lock the film-facing version of a character for:
- visual consistency
- shot planning
- performance direction
- prompt consistency
- continuity during generation

## Source-of-truth rule

For character truth:
- `story/characters/<name>/character.md` and `story/characters/<name>/summary.md` remain the canon reference

For film execution:
- the character bible holds only what the film pipeline needs to stay visually and emotionally consistent

## Keep these docs lean

A character bible should focus on:
- what the character must feel like on screen
- what the character should look like in the current film scope
- what should stay stable across generations
- what should not drift

Avoid turning these docs into duplicate lore summaries.

## Update rule

Only update a character bible when one of these changes:
- the approved visual look
- the acting/performance read
- the current production scope
- a story change directly affects how the character must be shown on screen

If a story beat changes but the character's visual/performance rules do not, the bible does not need an update.
